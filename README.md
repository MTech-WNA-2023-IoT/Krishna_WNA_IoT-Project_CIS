embedded systems, wireless networking, RF/SDR, image processing, materials-oriented analysis, and battery-health analytics.

## Important use note
These repositories are meant to be uploaded as **public proof-of-work**. They are intentionally:
- original and recreated from scratch,
- safe for public sharing,
- free from confidential project details,
- aligned with actual skills and experience.

Do **not** describe any of these as direct code from confidential defence or employer environments.
A better description is:

> “These are reconstructed portfolio projects inspired by the engineering domains I have worked in.”
